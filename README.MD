# Thirsty Drink Finder - Spiral

Developed using Next.js + TypeScript. Added simple unit tests. No UI library.

## Available Scripts

### Running the development server.

```bash
    yarn dev
```

### Building for production.

```bash
    yarn build
```

### Running the production server.

```bash
    yarn start
```

### Starting Unit test

```bash
    yarn test
```
